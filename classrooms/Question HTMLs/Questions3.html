Computer Science Unit 3 Questions:

Question: What is the function of a NOT gate?
Answer:A NOT gate inverts its input signal; if the input is 1 (true), the output is 0 (false), and vice versa.
Question: How do you construct a truth table for an AND gate?
Answer: An AND gate outputs 1 only if all its inputs are 1. For two inputs, the truth table shows an output of 1 only when both inputs are 1.
Question: Demonstrate how to convert an integer value from binary to denary representation
Answer: To convert from binary to denary, multiply each binary digit by 2 raised to the power of its position and sum the results.
Question: What is unique about the XOR (Exclusive OR) gate compared to the OR gate?
Answer: An XOR gate outputs 1 when an odd number of inputs are 1, unlike an OR gate, which outputs 1 when at least one input is 1.
Question: How is character data represented in its internal binary form, considering different character sets
Answer: Character data is represented in binary using character sets like ASCII, extended ASCII, or Unicode, where each character is assigned a unique binary code.
Question: How can you construct a logic circuit from a problem statement?
Answer: Identify the inputs and required output, determine the logical operations needed (using AND, OR, NOT, etc.), and then arrange the logic gates to achieve the desired output based on those conditions.r.
Question: Explain how data for a vector graphic is encoded
Answer: Data for a vector graphic is encoded using drawing objects, properties, and drawing lists, representing the geometric shapes and attributes of the image.
