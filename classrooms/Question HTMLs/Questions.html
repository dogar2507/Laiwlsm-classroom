Computer Science Unit 1 Questions:

Question: Explain the difference between "kibi" and "kilo" in the context of binary magnitudes
Answer: "Kibi" represents 2^10 (1024), while "kilo" represents 10^3 (1000) in the binary and decimal systems, respectively.
Question: Describe the significance of binary, denary, and hexadecimal number bases
Answer: Binary is base-2, denary is base-10, and hexadecimal is base-16. They are different numeral systems used for representing numbers in computing.
Question: Demonstrate how to convert an integer value from binary to denary representation
Answer: To convert from binary to denary, multiply each binary digit by 2 raised to the power of its position and sum the results.
Question: Explain the concept of overflow in the context of binary arithmetic
Answer: Overflow occurs when the result of a binary operation exceeds the maximum representable value, causing inaccuracies or errors.
Question: How is character data represented in its internal binary form, considering different character sets
Answer: Character data is represented in binary using character sets like ASCII, extended ASCII, or Unicode, where each character is assigned a unique binary code.
Question: Define the terms pixel, file header, image resolution, and colour depth/bit depth
Answer: A pixel is the smallest unit in an image, a file header contains metadata, image resolution is the number of pixels in an image, and colour depth/bit depth is the number of bits used to represent each pixel's color.
Question: Explain how data for a vector graphic is encoded
Answer: Data for a vector graphic is encoded using drawing objects, properties, and drawing lists, representing the geometric shapes and attributes of the image.
Question: Describe the representation and encoding of sound, including terms like sampling, sampling rate, and sampling resolution
