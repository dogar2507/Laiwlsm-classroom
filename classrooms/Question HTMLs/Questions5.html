Computer Science Unit 5 Questions:

Question: Why does a computer system require an Operating System?
Answer:An OS is essential for managing computer hardware resources and providing common services for computer programs, facilitating a user interface and enabling application software to be run on the computer.
Question: What are the key management tasks carried out by an Operating System?
Answer: Memory management, file management, security management, hardware management (including input/output and peripherals), and process management.
Question: What threats to computer and data security are posed by networks and the internet?
Answer:Malware (viruses, spyware), hackers, phishing, and pharming are significant threats.
Question:How can the risks posed by these threats be restricted?
Answer: Through security methods like encryption, setting access rights, using firewalls, and implementing secure authentication techniques.
Question: HHow do data validation and data verification protect the integrity of data?
Answer: Validation ensures that data input meets predefined criteria (e.g., range, format), while verification checks that data has been accurately transferred or entered.
Question: How can you construct a logic circuit from a problem statement?
Answer: Identify the inputs and required output, determine the logical operations needed (using AND, OR, NOT, etc.), and then arrange the logic gates to achieve the desired output based on those conditions.r.
What are some methods of data validation and verification?
Answer:Validation methods include range checks, format checks, and presence checks. Verification during data entry can involve visual checks and double entry, while data transfer verification might use parity checks and checksums
