Computer Science Unit 4 Questions:

Question:What is the relationship between assembly language and machine code?
Answer: Assembly language is a more human-readable form of machine code. Each assembly language instruction corresponds to a specific machine code operation.
Question: What is immediate addressing mode?
Answer:Immediate addressing mode allows an instruction to specify the actual data or operand within the instruction itself.
Question: How does a two-pass assembler function?
Answer: In the first pass, it scans the code to identify all labels and calculate their addresses. In the second pass, it translates the assembly instructions into machine code, using the addresses defined in the first pass.
Question:What is the purpose of the CMP instruction in assembly language?
Answer:The CMP instruction compares two values and sets the condition flags in the status register based on the result, which can then influence conditional branch instructions.
Question:How are bitwise AND, OR, and XOR operations useful in bit manipulation?s
Answer: These operations allow for the manipulation of individual bits within a byte or word, enabling tasks such as setting, clearing, toggling, or testing specific bits, which is useful for controlling hardware devices or managing data at a low level.
Question: Explain how data for a vector graphic is encoded
Answer: Data for a vector graphic is encoded using drawing objects, properties, and drawing lists, representing the geometric shapes and attributes of the image.
